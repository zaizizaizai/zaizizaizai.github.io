clox

(持续更新中，目前到scanner)

clox是由Bob Nystrom基于ｃ编写的编译器，最终解释结果编译成字节码，通过编写学习，可以了解编译器的基本原理
主流程主要分为三步：

  1. 扫描
  2. 编译
  3. 执行(虚拟机)

结构

chunk结构


  1. 分配具有更多容量的新数组
  2. 将现有元素从旧数组复制到新数组
  3. 更新capacity
  4. 删除旧数组
  5. 更新code指向新数组
  6. 既有空间，也可将元素储存在新数组中
  7. 更新cout

函数以及参数说明

reallocate函数

传递两个参数给reallocate()控制要实现的操作

oldsize newSize 操作
0 非0 　分配新块
非0 0 重置为空
非0 < oldSize 缩小现有分配
非0 > oldSize 增加现有分配

OP_CONSTANT

OP_CONSTANT采用单字节操作数制定从块的常量数组加载哪个常量
